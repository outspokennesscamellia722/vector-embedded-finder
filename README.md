## 🧭 What this app does

Vector Embedded Finder helps you search your local memory in a fast, simple way. It uses semantic search, so you can look for ideas, notes, files, and past content by meaning, not just exact words.

It is built for Windows and works with local data on your own computer. It also supports multimodal search, which means it can work with text and other content types tied to your local memory set.

## 🚀 First-time setup

After you open the app for the first time, set up the main parts it needs:

- **Embedding source**: Gemini Embedding 2
- **Vector database**: ChromaDB
- **Search scope**: local files, notes, or imported content
- **Raycast extension**: optional, for quick access from your workflow

If the app offers a scan or index button, use it to build your local memory set before searching.

## 🔎 How to use it

1. Open the app.
2. Let it finish indexing your data.
3. Type a search term in the search box.
4. Review the matches that appear.
5. Open the item that fits your need.

Try search terms like:

- a topic from a note
- a project name
- a person’s name
- a phrase you remember part of
- a task you worked on before

Because the app uses semantic search, it can find related results even when the exact words do not match.

## 🧩 Main features

### 🧠 Semantic search
Find content by meaning instead of exact text. This helps when you remember the idea but not the full phrase.

### 🗂️ Local memory storage
Keep your indexed content on your own computer. That makes it easier to search your personal data in one place.

### 🖼️ Multimodal support
Search across content tied to more than plain text, based on the way your local memory is set up.

### ⚡ Fast retrieval
ChromaDB helps the app return results quickly after indexing.

### 🪄 Raycast extension support
Use the extension to launch searches from Raycast if you want a quicker workflow on macOS-style launch tools in a cross-platform setup.

## ❓ Troubleshooting

### The app will not open
- Right-click the file and choose Run as administrator
- Check that the file is not still inside the zip
- Re-download the release if the file looks damaged

### Search returns no results
- Make sure indexing has finished
- Confirm the correct folder or source is loaded
- Try a broader search term

### The app asks for an API key
- Paste your Gemini API key into the setup field
- Save the setting before restarting the app

### The app feels slow
- Close other heavy apps
- Use a smaller data set first
- Check that your PC has enough RAM free
